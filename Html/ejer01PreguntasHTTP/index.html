Sección de Preguntas sobre HTTP

  • Sobre que protocolo binario está montado el protocolo HTTP?
    • Está montado sobre el protocolo de transporte TCP

  • Cuáles son los clientes http y los servidores http más utilizados?
    • Respuesta

  • Que verbos admite un comando en el requerimiento HTTP?
    • Se usan en los web services:GET, PUT, DELETE, POST y CONNECT (Este último lo usan los proxys)

  • Que contenido lleva el body de un requerimiento HTTP?
    • Puede contener datos en cualquier formato, por ejemplo en un POST tendría los datos ingresados en los campos de un formulario

  • Que diferencia existe entre un URL y un URI?
    • La URL es la ip de destino configurado en el servidor de nombres. La URI es el path hasta el recurso.

  • Cómo se almacena información relacionada con las respuestas http?
    • Se almacenan los datos de la resuestas en una cookie.

  • Que significa HTTP sobre conexiones preexistentes?
    • Puede reutilizar conexiones TCP para el asincronismo del HTTP

  • Que significa Virtual Hosting?
    • Respuesta

  • Que significa Cache por etag?
    • Respuesta

  • Porque el HTTP se considera STATE LESS?
    • Porque no puede hacer un pedido y tener unas determinadas variables en el servidor y luego hacer otra y pretender tener las mismas variables cambiadas.

  • Que nueva versión de HTTP se está usando para mejorar la velocidad WEB?
    • HTTP/2, utiliza misma semántica que HTTP1.1 compactando cabeceras, priorizando elementos a transferir y multiplexando sin esperar respuestas.

  • Cómo va a mejorar el comportamiento de una aplicación WEB en ambientes ruidosos o de débil conexión?
    • HTTP/3 utilizando QUIC
