Declaraciones en bloque definidas entre etiquetas style

h1{
font-size: 2em;
color: darkblue;
text-align: center;
background-color: aquamarine;
}

Todos los textos con etiquetas h1 tienen este estilo.

La etiqueta h1 ocupa todo el ancho del contenedor (por default lleva display en bloque)

La etiqueta h1 lleva texto de color azul

El background de un h1 lleva color aquamarine

El estilo definido para la etiqueta h1 indica que el texto ira centrado

p{
text-align: left;
font-size: 1em;
color: blue;
font-style: italic;
border-top-style: solid;
border-bottom-style:solid;
border-width: thin;
border-color: blue;
}

Todos los textos con etiquetas p tienen el estilo definido dentro de la etiqueta style

Para los párrafos se define un estilo con estas características: Alineación a la izquierda

Tamaño de letra de la misma medida que la definida para el contenedor body

Color de la letra azul

Estilo de la fuente italico

Bordes superior e inferior solidos, finos y de color azul
